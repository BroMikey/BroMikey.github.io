LCM(最小公倍数)

使用公式LCM(a, b) = |a * b| / GCD(a, b)
然后使用辗转相除法球的最大公约数

  1. b = 0, GCD = a;
  2. a = b, b = a % b, 递归至b = 0; 可以看作是一个长方形，长边是a，短边是b。如何使用一个正方形将该长方形填满，所求的最大的正方形边长就是a和b的最大公约数。
