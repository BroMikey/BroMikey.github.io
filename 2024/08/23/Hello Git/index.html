Hello Git

  • 设置用户信息
  • 初始化一个仓库
  • 将文件添加到缓存区
  • 提交更改
  • 查询提交记录，查询当前仓库文件状态
  • 分支创建管理

Git 采用 Merge（合并）的方式合并两个分支。使用该方法将分支 B 并入分支 A 时，会将 B 分支的所有 commit 并入 A 分支的提交历史中

使用 Squash 方式将分支 B 并入分支 A 时，在 B 分支上的所有更改会被合并为一次 commit 提交到 A 分支

使用 Rebase 方式将分支 B 并入分支 A 时，在 B 分支上的每一次 commit 都会单独添加到 A 分支
